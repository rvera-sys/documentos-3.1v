# 📋 DOCUMENTOS 3.1

Generador de Documentos Legales Inmobiliarios - Sistema Completo

## 🚀 Quick Start

```bash
# 1. Configurar variables de entorno
cp .env.example .env.local

# 2. Instalar dependencias
npm install

# 3. Crear Supabase (ver docs/SETUP_COMPLETO.md)
# 4. Deploy a Vercel

# 5. Frontend en vivo: https://tu-user.github.io/documentos-3-1
# 6. Backend en vivo: https://documentos-3-1.vercel.app
```

## 📋 Características

✅ 17 templates de documentos legales  
✅ 200+ cláusulas dinámicas  
✅ Google OAuth2 autenticación  
✅ Generación de PDFs  
✅ Versionado e historial  
✅ Panel de administración  
✅ 100% gratuito ($0/mes)  

## 🛠️ Stack

- **Frontend:** HTML/CSS/JavaScript (GitHub Pages)
- **Backend:** Vercel Functions (Node.js)
- **Database:** Supabase (PostgreSQL)
- **Auth:** Google OAuth2
